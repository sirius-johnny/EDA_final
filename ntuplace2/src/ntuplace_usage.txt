NLP arguments

-nlp  run nlp mode
-N    target_block
-d    target_density
-n    target_nnb
-w    wire weight
-i    inc factor
-R    cluster ratio


// Jin

-bbcellswap				run branch and bound cell swapping (default: closed)
-bbwindowsize [num]		window size (# of cells) used in bbcellswap (default: 3)
-bboverlapsize [num]	window overlap width (# of cells) used in bbcellswap (default: 2)
-bbiteration [num]		# of iterations for bbcellswap (default: 2)

// indark

-macrolegal 		Run macro legalizer after global placement	
			NOTE: macros will be fixed 
-macroheight [int]	Specify the max. # of row height a std. cell should be.
			DEFAULT: 1 (that is a module with the height more than 2 rows will be 
					treated as a macro)
